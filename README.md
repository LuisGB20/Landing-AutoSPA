# 🚗 AutoSPA Landing Page

Landing page moderna y responsive para un autolavado, diseñada con un enfoque en la captación de clientes. Su objetivo principal es convertir visitantes en prospectos mediante llamadas a la acción (CTA) que dirigen al usuario a una conversación de WhatsApp.

## ✨ Características

- Diseño moderno y responsive.
- Hero section orientado a la conversión.
- Sección de servicios con precios y características.
- Beneficios del negocio.
- Proceso de atención explicado paso a paso.
- Testimonios de clientes.
- Modal de contacto con formulario.
- Integración con WhatsApp mediante mensaje predefinido.
- Optimización básica para SEO.

## 🚀 Tecnologías

- HTML
- CSS Tradicional
- Tailwind CSS
- JavaScript

## 📱 Funcionamiento

1. El usuario navega por la landing.
2. Hace clic en cualquier llamada a la acción.
3. Se abre un formulario para recopilar sus datos.
4. Al enviar el formulario, se genera un mensaje personalizado y se redirige automáticamente a WhatsApp.
5. El negocio recibe un contacto listo para iniciar la conversación.

## 🎯 Objetivo

Esta plantilla está orientada a negocios que buscan incrementar la captación de clientes reduciendo la fricción en el proceso de contacto, utilizando WhatsApp como principal canal de comunicación.
